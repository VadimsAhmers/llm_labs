# Мультиагентный помощник по учёбе

Система из нескольких специализированных агентов для помощи с задачами по программированию, теории и планированию учёбы.

## Установка

```bash
pip install -r requirements.txt
```

Настройка

Создайте .env файл:

`UNIVERSITY_API_BASE=http://your-vllm-server:8000/v1
UNIVERSITY_API_KEY=your-key
LLM_MODEL=qwen3-32b`
